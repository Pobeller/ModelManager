# ModelManager

## Übersicht
ModelManager ist ein .NET 8 Projekt, das entwickelt wurde, um die Verwaltung und Bearbeitung von Datenmodellen zu erleichtern. Es bietet eine Reihe von Funktionen, die es Entwicklern ermöglichen, Datenmodelle effizient zu erstellen, zu aktualisieren und zu verwalten.

ModelManager ist ein Programm, um Diffusionsmodelle zu verwalten, herunterzuladen und einfach zu benutzen.
